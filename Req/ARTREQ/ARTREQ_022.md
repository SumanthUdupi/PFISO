# Art Asset Requirement: ARTREQ_022

## 1. Asset Title
**Modern Laptop**

## 2. Asset Type
Prop / Interactive (Hero)

## 3. Detailed Description
A sleek, aluminum unibody laptop. It is open on the desk, displaying code or the game's menu. This is the central "computer" reference in the office.

## 4. Style Guidelines
- **Aesthetic:** Premium Tech.
- **Material:** Anodized Aluminum.

## 5. Technical Specifications
- **Dimensions:** 15-inch form factor.
- **File Format:** FBX
- **Polycount:** ~500 tris.
- **Color Palette:**
  - Body: #cfcfcf (Silver) or #2d2424 (Space Grey).
  - Screen: Emissive Texture.

## 6. Lighting & Color Theme
- **Screen Glow:** The screen is a significant light source for the desk area (cool blue light vs warm lamp light).
- **Reflection:** Screen glass reflects the window.

## 7. Visual References
- MacBook Pro 16.

## 8. Integration Notes
- Screen texture will be a separate material index to allow swapping "screens" (e.g., VS Code, Desktop, Game Menu).
- "Lid" bone for closing animation.

## 9. Portfolio Impact
Hard surface modeling and emissive lighting integration.

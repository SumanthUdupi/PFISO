# Art Asset Requirement: ARTREQ_043

## 1. Asset Title
**Night Mode Lighting Profile**

## 2. Asset Type
Lighting / Config

## 3. Detailed Description
A complete alternative lighting setup for "Night Mode". The sun (Directional Light) is disabled or turned to moonlight (blue). The Desk Lamp (ARTREQ_034) and Monitor (ARTREQ_035) become the primary light sources.

## 4. Style Guidelines
- **Aesthetic:** Deep Focus / Late Night Coding.
- **Contrast:** High.

## 5. Technical Specifications
- **Sun Intensity:** 0 or 0.1 (Moon).
- **Ambient Color:** Dark Blue/Indigos.

## 6. Lighting & Color Theme
- **Cozy:** The "dark room, warm lamp" aesthetic is the peak of cozy.

## 7. Visual References
- Lofi Girl Night stream.

## 8. Integration Notes
- State machine switch.

## 9. Portfolio Impact
Dynamic lighting states.

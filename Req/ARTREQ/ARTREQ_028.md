# Art Asset Requirement: ARTREQ_028

## 1. Asset Title
**Flip-Style Desk Calendar**

## 2. Asset Type
Prop / Interactive

## 3. Detailed Description
A minimal calendar showing the date. Can be used to show the actual system date or a specific "launch date".

## 4. Style Guidelines
- **Aesthetic:** Type-driven.
- **Font:** Inter or similar sans-serif.

## 5. Technical Specifications
- **Dimensions:** Small desk footprint.
- **File Format:** FBX
- **Polycount:** ~100 tris.
- **Color Palette:**
  - Text: #2d2424.
  - Background: #fffcf5.

## 6. Lighting & Color Theme
- **Clean:** High contrast for readability.

## 7. Visual References
- Stendig Calendar (mini version).

## 8. Integration Notes
- Dynamic texture for date display.

## 9. Portfolio Impact
Dynamic text integration.

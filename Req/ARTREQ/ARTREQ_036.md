# Art Asset Requirement: ARTREQ_036

## 1. Asset Title
**Draped Fairy Lights**

## 2. Asset Type
Lighting / Prop

## 3. Detailed Description
A string of small LEDs draped over the bookshelf or window. Pure "cozy" aesthetic bait.

## 4. Style Guidelines
- **Aesthetic:** Dorm / Cozy Room.
- **Glow:** Soft, non-directional.

## 5. Technical Specifications
- **Dimensions:** 2m string.
- **File Format:** FBX (Cable) + Sprites (Glows).
- **Polycount:** ~400 tris.
- **Color Palette:**
  - Light: #fefae0 (Warm White).

## 6. Lighting & Color Theme
- **Bokeh:** Should create nice bokeh circles in depth of field background.

## 7. Visual References
- Pinterest "Cozy Desk Setup".

## 8. Integration Notes
- Decorative only, low light contribution to scene GI.

## 9. Portfolio Impact
Simulating complex light sources efficiently.

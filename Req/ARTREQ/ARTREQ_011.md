# Art Asset Requirement: ARTREQ_011

## 1. Asset Title
**Modern Interior Door - White Oak**

## 2. Asset Type
Environment / Structural

## 3. Detailed Description
A standard interior door with a modern, flat-panel design. It acts as the exit/entrance to the office space. It should include a simple lever handle. It typically stays closed but implies a connection to the rest of the house.

## 4. Style Guidelines
- **Aesthetic:** Clean, Solid.
- **Finish:** Matte paint or light wood veneer.
- **Hardware:** Brushed nickel or matte black.

## 5. Technical Specifications
- **Dimensions:** 0.9m x 2.1m.
- **File Format:** FBX
- **Polycount:** ~200 tris.
- **Color Palette:**
  - Door: #fefae0 (Off-white)
  - Handle: #4a403a (Dark Iron)

## 6. Lighting & Color Theme
- **Ambient Occlusion:** Door jambs need nice soft shadows to ground the door in the wall.

## 7. Visual References
- Modern flush doors.
- Standard US interior doors.

## 8. Integration Notes
- Placed in a wall slot.
- Handle should be a separate mesh if animation is ever needed (not required for MVP).

## 9. Portfolio Impact
Standard architectural asset execution.

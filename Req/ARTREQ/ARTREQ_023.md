# Art Asset Requirement: ARTREQ_023

## 1. Asset Title
**Wirebound Notebook (Open)**

## 2. Asset Type
Prop / Interactive

## 3. Detailed Description
An open notebook with a wire spiral binding. The pages have scribbles, sketches, or a to-do list visible.

## 4. Style Guidelines
- **Aesthetic:** Analog productivity.
- **Paper:** Off-white / Cream.

## 5. Technical Specifications
- **Dimensions:** A5 size.
- **File Format:** FBX
- **Polycount:** ~300 tris (for wire detail).
- **Color Palette:**
  - Cover: #3a5a40 (Forest Green).
  - Paper: #fffcf5.

## 6. Lighting & Color Theme
- **Texture:** Paper normal map should show paper grain.

## 7. Visual References
- Moleskine notebooks.

## 8. Integration Notes
- Text on page can be a decal or texture swap to show "current objectives".

## 9. Portfolio Impact
Texturing skills (handwriting).

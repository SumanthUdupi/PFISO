# Art Asset Requirement: ARTREQ_046

## 1. Asset Title
**Interaction Prompt Bubble**

## 2. Asset Type

## 3. Detailed Description

- **Symbol:** "E" key or Hand icon.

## 6. Lighting & Color Theme
- **Color:** #ffffff background, #4a403a icon.

## 7. Visual References
- Genshin Impact interact prompts.

## 8. Integration Notes

## 9. Portfolio Impact

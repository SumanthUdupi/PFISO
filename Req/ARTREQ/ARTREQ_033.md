# Art Asset Requirement: ARTREQ_033

## 1. Asset Title
**Sunlight Beam (God Rays)**

## 2. Asset Type
Lighting / Atmosphere

## 3. Detailed Description
A stylized volumetric light shaft entering through the main window. It visually defines the direction of the key light and creates that dusty, cozy atmosphere.

## 4. Style Guidelines
- **Aesthetic:** Dreamy / Ethereal.
- **Type:** Volumetric mesh or shader effect.

## 5. Technical Specifications
- **Dimensions:** Cone shape matching window frame.
- **File Format:** Custom Shader / Mesh.
- **Load:** Low (use simple alpha soft particles or mesh gradient).
- **Color Palette:**
  - #fffcf5 with Alpha 0.1 -> 0.0

## 6. Lighting & Color Theme
- **Warmth:** Adds the "golden hour" feel instantly.

## 7. Visual References
- "The Witness" lighting.
- "Animal Crossing" morning light.

## 8. Integration Notes
- Must fade out over distance.
- Must not occlude gameplay objects too heavily.

## 9. Portfolio Impact
Technical Art: Creating atmosphere without expensive real-time volumetrics.

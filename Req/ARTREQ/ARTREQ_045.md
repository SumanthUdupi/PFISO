# Art Asset Requirement: ARTREQ_045

## 1. Asset Title
**Cozy Custom Cursor**

## 2. Asset Type
UI Element

## 3. Detailed Description
A custom cursor to replace the standard OS pointer. Aligns with the pixel/retro font used in the game (`Press Start 2P`).

## 4. Style Guidelines
- **Aesthetic:** Pixel Art.
- **Shape:** Round glove or thick arrow.

## 5. Technical Specifications
- **Format:** .PNG / .CUR.
- **Size:** 32x32px.
- **Color:** #d4a373 (Orange) with White outline.

## 6. Lighting & Color Theme
- **Visibility:** Must be visible on both light and dark backgrounds (needs outline).

## 7. Visual References
- Retro RPG cursors.

## 8. Integration Notes
- CSS `cursor: url(), auto;`

## 9. Portfolio Impact
UI consistence and polish.

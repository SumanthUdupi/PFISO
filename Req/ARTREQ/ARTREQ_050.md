# Art Asset Requirement: ARTREQ_050

## 1. Asset Title
**"Deep Focus" Night Palette**

## 2. Asset Type
Color Theme

## 3. Detailed Description
The master reference for the Night/Dark mode. Inverts the brightness hierarchy while maintaining hue harmony.

## 4. Style Guidelines
- **Tone:** Quiet, Solitary, Calm.
- **Contrast:** High (Light pools vs Dark shadows).

## 5. Technical Specifications
- **Primary:** #2d2424 (Dark Espresso/Black).
- **Secondary:** #3a5a40 (Dark Forest).
- **Light:** #ffcc99 (Incandescent Lamp).
- **Accent:** #795548 (Muted Brown).

## 6. Lighting & Color Theme
- **Blue Shift:** Shadows shift towards teal/blue to contrast with orange lamps.

## 7. Visual References
- Dark Mode IDE themes (Gruvbox Dark).

## 8. Integration Notes
- CSS Dark mode overrides.

## 9. Portfolio Impact
Accessibility and aesthetic versatility.

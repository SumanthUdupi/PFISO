# REQ-037: Virtual Stick Tuning

## 1. Requirement Title
Virtual Stick Floating Pivot

## 2. Problem Statement
Fixed virtual sticks are hard to hit.

## 3. Proposed Solution
Implement "Floating Joystick". The stick center appears wherever the player first touches the relevant screen zone.

## 4. Implementation Details
- **File**: `MobileControls.tsx`
- **Logic**: On `touchstart`, set `stickPos` to `touchPos`.

## 5. Code Snippets
```typescript
<nipplejs-react options={{ dynamicPage: true }} />
```

## 6. Expected Benefits
- **UX**: Reduces "missed" inputs.

## 7. Dependencies
- CSS / UI Lib

## 8. Priority Level
**Medium**

## 9. Estimated Complexity
**Simple**

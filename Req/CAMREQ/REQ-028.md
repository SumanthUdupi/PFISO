# REQ-028: Dialog Camera Framing

## 1. Requirement Title
Automatic Dialog Framing (Shot-Reverse-Shot)

## 2. Problem Statement
Dialogs happen in standard view, which is impersonal.

## 3. Proposed Solution
When entering dialog, smooth transition the camera to an "Over the Shoulder" view of the speaker.

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **Logic**: Calculate position: `SpeakerPos + SpeakerForward * 1.5 + Up * 1.5`. Lerp to it.

## 5. Code Snippets
```typescript
const target = speaker.position.clone().add(offset);
camera.position.lerp(target, dt * 2);
```

## 6. Expected Benefits
- **Narrative**: Cinematic quality.

# REQ-022: Strafe Tilt

## 1. Requirement Title
Camera Roll on Strafe

## 2. Problem Statement
Lateral movement feels stiff.

## 3. Proposed Solution
Rotate the camera slightly on the Z-axis (Roll) when moving left or right.

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **Logic**: `targetRoll = -inputX * maxRollAngle`. Damp to target.

## 5. Code Snippets
```typescript
const tilt = moveInput.x * -0.05; // -0.05 rads
camera.rotation.z = THREE.MathUtils.lerp(camera.rotation.z, tilt, dt * 5);
```

## 6. Expected Benefits
- **Feel**: Adds weight and momentum to movement.

## 7. Dependencies
- None

## 8. Priority Level
**Medium**

## 9. Estimated Complexity
**Simple**

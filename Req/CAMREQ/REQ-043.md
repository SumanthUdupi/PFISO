# REQ-043: Object Hiding (Occlusion Culling)

## 1. Requirement Title
Portal/Zone Based Occlusion Culling

## 2. Problem Statement
In a room-based game (Lobby, Office), rendering the outside world when in the lobby wastes resources.

## 3. Proposed Solution
If Camera is in Zone A, disable rendering of Zone B (if not visible).

## 4. Implementation Details
- **File**: `Zones.tsx` / `CameraSystem.tsx`
- **Logic**: Check Camera position against Zone AABB. Set `zoneRoot.visible = false` for others.

## 5. Code Snippets
```typescript
if (!cameraBounds.intersectsBox(zoneBox)) {
    zone.visible = false;
}
```

## 6. Expected Benefits
- **FPS**: Massive boost in complex scenes.

## 7. Dependencies
- Zone System

## 8. Priority Level
**Low**

## 9. Estimated Complexity
**Complex**

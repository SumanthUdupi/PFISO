# REQ-045: Memory Management for Camera Assets

## 1. Requirement Title
Disposal of Post-Processing Buffers

## 2. Problem Statement
Switching scenes or effects might leak render targets.

## 3. Proposed Solution
Ensure `EffectComposer` and effects are properly disposed on unmount.

## 4. Implementation Details
- **File**: `PostProcessingEffects.tsx`
- **Logic**: standard `useEffect` cleanup.

## 5. Code Snippets
```typescript
return () => {
    composer.dispose();
}
```

## 6. Expected Benefits
- **Stability**: Prevents crashes on mobile.

## 7. Dependencies
- None

## 8. Priority Level
**Low**

## 9. Estimated Complexity
**Simple**

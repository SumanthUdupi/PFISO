# REQ-035: Chromatic Aberration on Impact

## 1. Requirement Title
Chromatic Aberration Spike on Impact

## 2. Problem Statement
Big hits or explosions feel flat.

## 3. Proposed Solution
Briefly increase Chromatic Aberration offset when a large "Shock" event occurs.

## 4. Implementation Details
- **File**: `PostProcessingEffects.tsx`
- **Logic**: Listen to `SCREEN_SHAKE`. Map intensity to `chromaticAberration.offset`. Decay over time.

## 5. Code Snippets
```typescript
useFrame((_, dt) => {
    chromaticRef.current.offset.x = lerp(current, 0, dt * 5);
})
```

## 6. Expected Benefits
- **Impact**: Glitchy/Digital feel for damage.

## 7. Dependencies
- None

## 8. Priority Level
**Medium**

## 9. Estimated Complexity
**Simple**

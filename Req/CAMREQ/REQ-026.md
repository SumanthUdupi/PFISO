# REQ-026: Lock-On Target System

## 1. Requirement Title
Target Lock-On Mechanism (Z-Targeting)

## 2. Problem Statement
Combat or Focus requires keeping an enemy/object in view while moving. Manual control is difficult.

## 3. Proposed Solution
Allow locking the camera focus onto a specific `Object3D`.

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **State**: `target: Object3D | null`
- **Logic**: If `target` exists, `camera.lookAt(target.position)`. Override manual rotation inputs.

## 5. Code Snippets
```typescript
if (lockedTarget) {
    const lookPos = lockedTarget.position.clone();
    camera.lookAt(lookPos);
    // Smoothly interpolate Quaternion for better feel
}
```

## 6. Expected Benefits
- **Gameplay**: Essential for combat depth.

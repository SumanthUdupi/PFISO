# REQ-014: Vertical Look Limits (Pitch Clamp)

## 1. Requirement Title
Configurable Vertical Look Limits (Pitch Clamping)

## 2. Problem Statement
Users can often look too far down (seeing inside the character) or too far up (flipping the camera).

## 3. Proposed Solution
Clamp the vertical angle (Theta/Phi) of the camera relative to the player.

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **Logic**: Convert Cartesian to Spherical, clamp Phi, convert back. Or use Euler clamping if using local rotation.

## 5. Code Snippets
```typescript
const minPolarAngle = Math.PI / 4; // 45 deg
const maxPolarAngle = Math.PI / 1.5; // 120 deg
spherical.phi = Math.max(minPolarAngle, Math.min(maxPolarAngle, spherical.phi));
```

## 6. Expected Benefits
- **Quality**: Prevents broken camera angles.
- **Control**: Keeps the view focused on relevant gameplay areas.

## 7. Dependencies
- None

## 8. Priority Level
**High**

## 9. Estimated Complexity
**Simple**

# REQ-003: Input System Integration

## 1. Requirement Title
Integrate Camera with Central Input System

## 2. Problem Statement
`Player.tsx` currently reads `joystick` from `useControlsStore` but also effectively hardcodes some input logic. Camera input (look vector) should be normalized across all devices (Mouse, Gamepad, Touch) via `InputManager`.

## 3. Proposed Solution
Ensure `InputManager` handles "Look" actions (Mouse Delta, Right Stick, Touch Drag) and feeds a normalized `Vector2` output to the `ControlsStore`. The `CameraSystem` should only read this normalized vector.

## 4. Implementation Details
- **File**: `src/systems/InputManager.ts`
- **Action**: Add `LOOK_X` and `LOOK_Y` bindings.
- **File**: `src/stores/controlsStore.ts`
- **Action**: Ensure `lookVector` is updated by `InputManager` every frame.

## 5. Code Snippets
```typescript
// InputManager
const getLookVector = () => {
   // Combine mouse delta, gamepad right stick, etc.
   return { x: ..., y: ... }
}
```

## 6. Expected Benefits
- **Consistency**: Unified input handling for all devices.
- **Configurability**: Easier to implement sensitivity/inversion later.

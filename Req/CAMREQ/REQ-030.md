# REQ-030: Screenshot Mode Enhancements

## 1. Requirement Title
Enhanced Photo Mode (Clean UI)

## 2. Problem Statement
Players want to take screenshots without UI.

## 3. Proposed Solution
A "Photo Mode" state that hides HUD, allows free camera movement (within bounds), and changes FOV/Depth of Field settings.

## 4. Implementation Details
- **File**: `CameraSystem.tsx` / `HUD.tsx`
- **Logic**: `useGameStore` flag `isPhotoMode`. `HUD` returns null if true. `CameraSystem` switches to `FREE` mode.

## 5. Code Snippets
```typescript
if (isPhotoMode) {
    // Enable free fly logic
}
```

## 6. Expected Benefits
- **Community**: Encourages sharing.

## 7. Dependencies
- REQ-004

**Medium**

## 9. Estimated Complexity

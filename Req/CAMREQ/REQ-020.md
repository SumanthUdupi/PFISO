# REQ-020: Touch Input Gestures (Pinch Zoom)

## 1. Requirement Title
Mobile Pinch-to-Zoom

## 2. Problem Statement
Mobile users have no easy way to adjust camera distance/FOV.

## 3. Proposed Solution
Detect 2-finger pinch gesture to adjust `cameraDistance`.

## 4. Implementation Details
- **File**: `MobileControls.tsx` or `InputManager.ts`
- **Logic**: Track distance between two touch points. Delta distance maps to Zoom In/Out command.

## 5. Code Snippets
```typescript
const dist = Math.hypot(t1.x - t2.x, t1.y - t2.y);
if (dist > prevDist) zoomIn();
```

## 6. Expected Benefits
- **UX**: Standard mobile gesture expectation.

## 7. Dependencies
- `useGesture` (hook library) or custom logic

## 8. Priority Level
**Medium**

## 9. Estimated Complexity
**Moderate**

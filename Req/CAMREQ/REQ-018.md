# REQ-018: Gamepad Stick Deadzones

## 1. Requirement Title
Radial Deadzone for Gamepads

## 2. Problem Statement
Gamepad sticks often have "drift" where they report small non-zero values even when neutral. This causes the camera to slowly rotate on its own.

## 3. Proposed Solution
Implement a radial deadzone (circular). Ignore inputs with magnitude less than `DEADZONE_THRESHOLD` (e.g., 0.15).

## 4. Implementation Details
- **File**: `InputManager.ts`
- **Logic**: Calculate vector length. If `< 0.15`, return `(0,0)`. If `> 0.15`, rescale remaining range `0..1` to avoid "jump" at 0.15.

## 5. Code Snippets
```typescript
const len = vec.length();
if (len < 0.15) return {x:0, y:0};
// Rescale
const norm = vec.normalize();
return norm.multiplyScalar((len - 0.15) / (1 - 0.15));
```

## 6. Expected Benefits
- **Quality**: Eliminates camera drift.

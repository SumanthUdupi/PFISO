# REQ-040: UI Safe Areas

## 1. Requirement Title
Camera Offset for UI Safe Areas

## 2. Problem Statement
UI elements (Action Buttons, Joystick) obscure the player character in center of screen on mobile.

## 3. Proposed Solution
Shift the camera's *projection offset* or target point so the character appears slightly "up" or "side" of center, in clear visual space.

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **Logic**: `camera.setViewOffset(...)` or just add `offsetY` to `target`.

## 5. Code Snippets
```typescript
if (isMobile) {
    target.y += 0.5; // Shift character up so it's not under thumb
}
```

## 6. Expected Benefits
- **Visibility**: Clear view of character.

## 7. Dependencies
- None

## 8. Priority Level
**Medium**

## 9. Estimated Complexity
**Simple**

# REQ-033: Motion Blur Intensity

## 1. Requirement Title
Camera Velocity-Based Motion Blur

## 2. Problem Statement
Motion blur acts on object movement, but sometimes camera rotation causes excessive blurring.

## 3. Proposed Solution
Tune Motion Blur parameters based on Camera Rotation Speed. If rotating fast, reduce blur length slightly to prevent "soup" visuals, or ensure per-object blur is prioritized.

## 4. Implementation Details
- **File**: `PostProcessingEffects.tsx`
- **Logic**: Adjust `MotionBlur` prop `intensity` inversely to `camera.rotationSpeed`? Or usually just enable it if missing.

```typescript
// N8AO or standard MotionBlur
```

- **Smoothness**: Hides low framerates.

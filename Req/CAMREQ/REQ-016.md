# REQ-016: Mouse Sensitivity Settings

## 1. Requirement Title
Configurable Mouse Sensitivity

## 2. Problem Statement
Sensitivity is hardcoded. Players with different mice (DPI) or preferences need to adjust this.

## 3. Proposed Solution
Multiply the mouse input delta by a user-configurable factor.

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **Formula**: `finalDelta = rawDelta * sensitivity * 0.001`

## 5. Code Snippets
```typescript
const sensitivity = useGameStore(s => s.sensitivity); // 0.1 to 10.0
const moveX = event.movementX * sensitivity * 0.002;
```

## 6. Expected Benefits
- **Accessibility**: Essential for standard PC gaming.

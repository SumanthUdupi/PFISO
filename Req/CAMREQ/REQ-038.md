# REQ-038: Auto-Center Camera

## 1. Requirement Title
Auto-Center Camera on Movement

## 2. Problem Statement
Mobile players struggle to move and look simultaneously.

## 3. Proposed Solution
If moving but NOT touching the look stick, slowly rotate camera behind the player.

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **Logic**: `if (moving && !looking) lerpRigRotation(...)`

## 5. Code Snippets
```typescript
const targetY = Math.atan2(moveDir.x, moveDir.z);
camera.rotation.y = lerpAngle(camera.rotation.y, targetY, dt * 0.5); // Slow
```

- **Accessibility**: Casual friendly.

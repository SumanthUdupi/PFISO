# REQ-021: Head Bobbing

## 1. Requirement Title
Procedural Head Bobbing (FPS)

## 2. Problem Statement
First Person mode feels like a floating camera ("camera on rails"). It lacks the physicality of walking.

## 3. Proposed Solution
Apply a sine wave offset to the camera Y and X position based on movement speed and time.

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **Formula**:
  - `y = sin(time * freq * 2) * amp`
  - `x = cos(time * freq) * amp * 0.5`

## 5. Code Snippets
```typescript
if (isMoving && isGrounded) {
    const bob = Math.sin(clock.elapsedTime * 10) * 0.05;
    camera.position.y += bob;
}
```

## 6. Expected Benefits
- **Immersion**: Feels like walking.

## 7. Dependencies
- None

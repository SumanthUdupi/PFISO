# REQ-049: Free Cam Mode (Developer)

## 1. Requirement Title
Noclip / Free Fly Debug Camera

## 2. Problem Statement
Developers need to inspect the level without physics constraints.

## 3. Proposed Solution
Toggle `OrbitControls` or a custom `FlyControls` on keypress (e.g. F8).

## 4. Implementation Details
- **File**: `App.tsx`
- **Logic**: `useGameStore((s) => s.devMode)`.

## 5. Code Snippets
```typescript
if (devMode) return <OrbitControls />;
return <CameraSystem />;
```

## 6. Expected Benefits
- **Dev Speed**: Faster level inspection.

## 7. Dependencies
- REQ-002

## 8. Priority Level
**Medium**

## 9. Estimated Complexity
**Simple**

# REQ-023: Landing Impact Shake

## 1. Requirement Title
Vertical Shake on Landing

## 2. Problem Statement
Landing from a jump feels floaty with no visual feedback.

## 3. Proposed Solution
Detect `isGrounded` transition. If vertical velocity was high, trigger a vertical "thud" shake (Screen Shake event).

## 4. Implementation Details
- **File**: `CameraSystem.tsx` / `Player.tsx`
- **Logic**: `if (justLanded && prevVelY < -5) emitShake(Math.abs(prevVelY) * 0.01)`

## 5. Code Snippets
```typescript
if (!wasGrounded && isGrounded) {
    // Landed
    triggerShake({ intensity: 0.5, direction: 'VERTICAL' });
}
```

## 6. Expected Benefits
- **Feedback**: Confirms landing, adds weight.

## 7. Dependencies
- `CameraShake` system

## 8. Priority Level
**Medium**

## 9. Estimated Complexity
**Simple**

# REQ-039: Portrait Mode FOV Adjustment

## 1. Requirement Title
Vertical FOV Correction for Portrait

## 2. Problem Statement
In Portrait mode, default horizontal FOV makes the view extremely narrow horizontally.

## 3. Proposed Solution
Increase FOV in Portrait mode to maintain horizontal situational awareness.

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **Logic**: `fov = isLandscape ? 40 : 65`

## 5. Code Snippets
```typescript
const isPortrait = size.height > size.width;
camera.fov = isPortrait ? 65 : 45;
```

## 6. Expected Benefits
- **UX**: Usable portrait gameplay.

## 7. Dependencies
- `useThree`

## 8. Priority Level
**Critical** (for mobile)

## 9. Estimated Complexity
**Simple**

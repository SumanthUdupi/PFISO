# REQ-024: Sprint FOV Effect

## 1. Requirement Title
Dynamic FOV Widening on Sprint

## 2. Problem Statement
Sprinting doesn't feel fast enough.

## 3. Proposed Solution
Widen the Field of View when sprinting.

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **Logic**: `targetFOV = isSprinting ? baseFOV + 10 : baseFOV`

## 5. Code Snippets
```typescript
camera.fov = lerp(camera.fov, targetFov, dt * 2);
camera.updateProjectionMatrix();
```

## 6. Expected Benefits
- **Sensation**: Classic "speed lines" effect.

## 7. Dependencies
- None

## 8. Priority Level
**High**

## 9. Estimated Complexity
**Simple**

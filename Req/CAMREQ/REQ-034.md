# REQ-034: Vignette on Damage

## 1. Requirement Title
Dynamic Red Vignette on Low Health

## 2. Problem Statement
Health bars are often ignored in heat of battle.

## 3. Proposed Solution
Pulse a red Vignette or Overlay when health is < 30%.

## 4. Implementation Details
- **File**: `PostProcessingEffects.tsx`
- **Logic**: Bind Vignette `darkness` and `color` to `GameStore.health`.

## 5. Code Snippets
```typescript
const health = useGameStore(s => s.health);
const darkness = health < 30 ? 0.8 : 0.4;
<Vignette darkness={darkness} eskil={false} />
```

## 6. Expected Benefits
- **Feedback**: Critical game state info.

## 7. Dependencies
- None

## 8. Priority Level
**High**

## 9. Estimated Complexity
**Simple**

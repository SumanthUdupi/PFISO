# REQ-029: Cutscene Spline Paths

## 1. Requirement Title
Catmull-Rom Splines for Cutscenes

## 2. Problem Statement
Need scripted camera movements for intros or events.

## 3. Proposed Solution
Support following a defined `CatmullRomCurve3`.

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **Data**: Array of `Vector3` points.
- **Logic**: `camera.position.copy(curve.getPointAt(t))`.

## 5. Code Snippets
```typescript
const point = spline.getPoint(progress);
camera.position.copy(point);
camera.lookAt(spline.getPoint(progress + 0.01));
```

## 6. Expected Benefits
- **Cinematography**: Professional intro sequences.

# REQ-019: Gamepad Acceleration Curves

## 1. Requirement Title
Response Curve (Exponent) for Gamepads

## 2. Problem Statement
Linear mapping of stick input feels "twitchy" for small adjustments and "too slow" for big turns.

## 3. Proposed Solution
Apply an exponential curve to the input. `Output = Input ^ Power`. Common power is 2.0 to 2.5.

## 4. Implementation Details
- **File**: `CameraSystem.tsx` or `InputManager.ts`
- **Logic**: `val = Math.sign(val) * Math.pow(Math.abs(val), 2.2)`

## 5. Code Snippets
```typescript
const curve = 2.0;
const x = Math.sign(rawX) * Math.pow(Math.abs(rawX), curve);
```

## 6. Expected Benefits
- **Precision**: Fine aiming near center.
- **Speed**: Fast turning at edges.

# REQ-005: Camera Configuration Store

## 1. Requirement Title
Create Camera Configuration Store (User Settings)

## 2. Problem Statement
Values like FOV, Sensitivity, and Inversion are hardcoded constants (e.g., `BASE_FOV = 40`). Users cannot customize their experience.

## 3. Proposed Solution
Create a persistent part of `useGameStore` or `useSettingsStore` to hold user preferences.
- `fovBase`
- `mouseSensitivity`
- `gamepadSensitivity`
- `invertY`
- `cameraDistance` (for third person)

## 4. Implementation Details
- **File**: `src/stores/settingsStore.ts` (create if needed) or `src/store.ts`
- **Integration**: `CameraSystem` reads these values instead of constants.

## 5. Code Snippets
```typescript
const { sensitivity, invertY } = useSettingsStore();
const deltaY = input.y * sensitivity * (invertY ? -1 : 1);
```

## 6. Expected Benefits
- **Accessibility**: Essential for many players.
- **Comfort**: Reduces motion sickness.

## 7. Dependencies
- None

## 8. Priority Level
**High**

## 9. Estimated Complexity
**Simple**

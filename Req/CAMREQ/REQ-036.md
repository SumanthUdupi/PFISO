# REQ-036: Gyroscope Look Assist (Mobile)

## 1. Requirement Title
Mobile Gyroscope Fine Aiming

## 2. Problem Statement
Aiming with touch sticks is imprecise.

## 3. Proposed Solution
Use Gyroscope (if available) to add small offsets to the view rotation. "Stick for broad turns, Tilt for fine aim."

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **Logic**: Use `DeviceOrientationEvent`. Map `beta`/`gamma` deltas to `rotation.x`/`y`.

## 5. Code Snippets
```typescript
if (isMobile && useGyro) {
    camera.rotation.y += gyroDelta.alpha * 0.05;
}
```

## 6. Expected Benefits
- **Accessibility**: Makes mobile play much closer to mouse quality.

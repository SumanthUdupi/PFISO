# REQ-047: Camera Parameter GUI

## 1. Requirement Title
Runtime Tuning GUI (Leva/Dat.GUI)

## 2. Problem Statement
Tweaking damping, stiffness, and offsets requires recompiling.

## 3. Proposed Solution
Expose all constants to Leva controls (only in Dev).

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **Lib**: `useControls` from Leva.

## 5. Code Snippets
```typescript
const { dampTime } = useControls({ dampTime: 0.2 });
```

## 6. Expected Benefits
- **Quality**: Perfect tuning by "feeling" it live.

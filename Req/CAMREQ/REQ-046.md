# REQ-046: Debug Visualization

## 1. Requirement Title
Camera Debug Gizmos (Collision Rays)

## 2. Problem Statement
Hard to debug why camera is clipping or jerking without seeing the collision checks.

## 3. Proposed Solution
Draw lines for Raycasts and Wireframes for SphereCasts when `debugFlags.showCamera` is true.

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **Logic**: Use `<Line>` or `three.js` helpers.

## 5. Code Snippets
```typescript
{debug && <Line points={[target, cameraPos]} color="red" />}
```

## 6. Expected Benefits
- **Dev Speed**: Faster tuning.

## 7. Dependencies
- REQ-001

## 8. Priority Level
**Medium**

## 9. Estimated Complexity
**Simple**

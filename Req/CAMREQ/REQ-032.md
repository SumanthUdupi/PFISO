# REQ-032: Depth of Field Focus

## 1. Requirement Title
Dynamic Depth of Field (Autofocus)

## 2. Problem Statement
Everything is always in focus, or focus is static.

## 3. Proposed Solution
Raycast to the center of the screen (Crosshair) to determine "Focus Distance". Smoothly interpolate the DOF `target` or `focusDistance` property.

## 4. Implementation Details
- **File**: `PostProcessingEffects.tsx` / `CameraSystem.tsx`
- **Logic**: `raycaster.setFromCamera`. `damp(dof.target, hit.distance, ...)`

## 5. Code Snippets
```typescript
<DepthOfField target={[0, 0, focusDist]} focalLength={0.02} bokehScale={2} height={480} />
```

## 6. Expected Benefits
- **Visuals**: Photographic quality.

## 7. Dependencies
- React Three Postprocessing

## 8. Priority Level
**Medium**

## 9. Estimated Complexity
**Moderate**

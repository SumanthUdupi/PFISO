# REQ-041: Camera Frustum Culling Check

## 1. Requirement Title
Optimization: Force Frustum Culling

## 2. Problem Statement
Some large objects or particles might be rendering even when off-screen.

## 3. Proposed Solution
Ensure `frustumCulled` is true for all meshes (default in Three.js), but verify specifically for custom shaders or particles.

## 4. Implementation Details
- **File**: `AssetLoader.ts`
- **Logic**: Traverse load result. `mesh.frustumCulled = true`.

## 5. Code Snippets
```typescript
scene.traverse(obj => {
    if (obj.isMesh) obj.frustumCulled = true;
});
```

## 6. Expected Benefits
- **FPS**: Higher performance.

## 7. Dependencies
- None

## 8. Priority Level
**Low**

## 9. Estimated Complexity
**Simple**

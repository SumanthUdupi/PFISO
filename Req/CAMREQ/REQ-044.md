# REQ-044: Dynamic Render Scale

## 1. Requirement Title
Dynamic Resolution Scaling (DPI)

## 2. Problem Statement
High DPI screens (Mobile/Retina) choke the GPU.

## 3. Proposed Solution
Adjust `dpr` (Device Pixel Ratio) based on performance. Start at 1.5, drop to 1.0 if FPS < 30.

## 4. Implementation Details
- **File**: `FPSLimiter.tsx` / `App.tsx`
- **Logic**: Monitor average FPS. Set `useThree(({ setDpr }) => setDpr(newVal))`.

## 5. Code Snippets
```typescript
if (avgFps < 30 && dpr > 1) {
    setDpr(1);
}
```

## 6. Expected Benefits
- **Performance**: Usable on low-end devices.

## 7. Dependencies
- `FPSLimiter`

## 8. Priority Level
**High**

## 9. Estimated Complexity
**Moderate**

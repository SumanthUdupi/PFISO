# REQ-017: Axis Inversion Settings

## 1. Requirement Title
Y-Axis and X-Axis Inversion

## 2. Problem Statement
Many players, especially those accustomed to flight sims or older games, prefer Inverted Y-Axis.

## 3. Proposed Solution
Boolean flags to invert the look direction.

## 4. Implementation Details
- **File**: `CameraSystem.tsx`
- **Logic**: `deltaY = rawDeltaY * (invertY ? -1 : 1)`

## 5. Code Snippets
```typescript
const invertY = useGameStore(s => s.invertY);
if (invertY) rotation.x -= deltaY; else rotation.x += deltaY;
```

## 6. Expected Benefits
- **Accessibility**: Standard feature.

## 7. Dependencies
- REQ-005

## 8. Priority Level
**High**

## 9. Estimated Complexity
**Simple**
